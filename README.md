# bbs_A_webchat
a BBS and webchat system based on Django

## 项目说明   
项目是基于Django实现的一个BBS和在线webchat的系统，主要实现了博文的管理（新增、分类等），评论，点赞以及webchat部分，目前功能可以发送消息给用户，用户组。 窗口切换，新消息提醒，聊天窗口拖动。 可以发文字、图片以及表情包。

## 依赖插件  
django-ckeditor   
Pillow
